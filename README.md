## ⚙️ Configuration

<details>
<summary><strong>🚀 FastFlag Configuration</strong></summary>

### Custom Presets
FastFlag configurations are stored in `FastFlagPresets.json`. You can:
- ✅ Modify existing presets
- ✅ Add custom presets  
- ✅ Import community presets
- ✅ Backup and restore configurations

### Supported Launcher Paths
| Launcher | Configuration Path |
|----------|-------------------|
| **Roblox** | `%LOCALAPPDATA%\Roblox\` |
| **Bloxstrap** | `%LOCALAPPDATA%\Bloxstrap\` |
| **Fishstrap** | `%LOCALAPPDATA%\Fishstrap\` |

</details>

## ⚠️ Important Considerations

<div align="center">

> **🛡️ Safety First - Always backup your Roblox data before making changes**

</div>

### 🔒 Safety Guidelines

| ⚠️ **Warning** | 📝 **Description** |
|----------------|-------------------|
| **Data Backup** | Always backup your Roblox folder before applying changes |
| **Stability Risk** | FastFlags may affect game stability - test thoroughly |
| **Detection Risk** | Some flags may be detected by anti-cheat systems |
| **Update Impact** | Keep tools updated for best compatibility |

### 🎯 Best Practices

- ✅ **Start with conservative presets** (Don's or Antwis flags)
- ✅ **Test in a safe environment** before competitive play
- ✅ **Keep original settings backed up**
- ✅ **Monitor performance after changes**
- ✅ **Stay updated** with latest tool versions

<details>
<summary><strong>🆘 Troubleshooting Guide</strong></summary>

### Common Issues & Solutions

**🔧 FastFlag Issues:**
- Clear existing flags if experiencing problems
- Try a different preset if one doesn't work
- Verify launcher path is correct

**🎨 CDBL-Lite Issues:**  
- Ensure textures are in supported formats
- Check file permissions for custom assets
- Restart Roblox after configuration changes

**⚙️ Fleasion Issues:**
- Verify Python installation and dependencies
- Check database permissions
- Review settings.json for configuration errors

</details>
